CSS History and Information

CSS Definition

CSS stands for Cascading Style Sheets. CSS makes the HTML wedbpages look good. CSS can control layout, colors, font - basically any element on the page can be changed from the default (browser) style with CSS.

History of CSS

  • CSS1 was released in 1996
  • CSS2 was released in 1998
  • Css3 - this is the current cersion - it is a living standard

Basic Structure of CSS

selector{ property: value; property2: value; }

  • CSS selectors can be an HTML element (example: p,body), id of an HTML element, custom class
  • CSS helps with the separation of content and looks
  • Easy to maintain
  • Accessibility issues can be more easily managed

There are 3 ways to include CSS on a webpage

  • External - this is the best practice - CSS is in a CSS file
  • Embedded - this applies to 1 page
  • Inline - this applies to a specific HTML element
